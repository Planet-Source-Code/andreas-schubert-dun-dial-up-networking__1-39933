Dial-up networking for Visual Basic

provided by: Andreas Schubert (eMail: andy@andreas-schubert.net)

Recently many people asked about using dial-up networking in their programs.

Fact is, there is not much information available about this, and it seemed to me that the questions were not answered sufficient enough. With my last project I had to deal with this problems too, so I decided to write a nifty class for dealing with dun.

Dial-up networking with VB is pretty easy, once you have translated all the required data structures for the api-functions (which I have done for you). The remaining part mostly consists of calling some functions and handling return values. For this, I created the small sample project.

Sample project:

 

 

Phonebook entries

Function fRasGetAllEntries

Retrieve a list of all dial-up connections on the machine

Function fRASRenameEntry

renames an existing dial-up connection

Function fRASDeleteEntry

deletes an existing dial-up connection

Function fRASValidateEntryName

 validating an entry

Function fRasGetEntryProperties

Returns detailed information about an existing entry

Function fRasSetEntryProperties

Set information for an existing entry / create new entry

 

 

 

 

Dialing information

(

Function fRasGetEntryDialParams

Get dial information for an existing entry

Function fRasSetEntryDialParams

Set dial information for an existing entry
Note: Phone numbers can only be set via SetEntryProperties!)

 

 

 

Dialing

Function fASyncronousDial

dial a entry syncronous

Function fSyncronousDial

dial a entry syncronous

Function fRasHangUp

finish an existing dial-up connection

 

 

Connection information

Function fRasGetConnectStatus

Returning status information about an existing connection

Function fRasEnumConnections

enumerate all connections and fill tRasconn array

 

 

Projections

Function fRasGetRASAMB

describes the result of a RAS Authentication Message Block

Function fRasGetRASPPPNBF

describes the result of a NetBeui projection

Function fRasGetRASPPPIPX

describes the result of a IIPX projection

Function fRasGetRASPPPIP

describes the result of a IP projection

Function fRasGetRASSLIP

describes the result of a serial line ip projection

 

 

 

 

Devices

Function fRasEnumDevices

returns a list of all RAS capable devices installed on the computer giving their name and devicetype

 

 

 Windows dialogs

Procedure sDLG_AddModem

displays the "Add New Modem" dialog

Procedure sDLG_ConfigModem

displays the "Configure Modem" dialog

Function fRASCreatePhoneBookEntryDialog

shows the "create new entry" dialog

Function fRASEditPhoneBookEntryDialog

shows the "edit phonebook entry" dialog

 

 

 

Other

Function fRasErrorHandler

returns a RAS errorcode as plaintext

Function fBytesToRasDialParams

Transform a bytearray into a tRASDialParams structure

Function fRasDialParamsToBytes
